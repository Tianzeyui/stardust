# 提示词 + 工具披露优化

## 1. 提示词从 4 段压到 2 段

- [ ] Rules+Verification+Safety 合并为一段（去重合并）
- [ ] Tools+Agent+Fork 合并为一段
- [ ] 目标：15+ 条规则 → 8 条以内

## 2. 工具披露——两阶段扩大到所有场景

- [ ] 当前：只有纯对话走 fast 无工具，编码场景仍然全量工具
- [ ] 改进：所有请求先 fast 无工具判断意图
  - 纯对话 → fast 直接回复
  - 编码 → pro + 工具
- [ ] 好处：减少工具 token 消耗 + 避免模型被无关工具干扰
