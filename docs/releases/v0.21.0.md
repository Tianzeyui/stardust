# BrainPlus v0.21.0

发布日期：2026-06-14

## 🏗 项目重构

- **重组项目结构**：brainPlusDesktop 提升为根目录，移除 brainPlusMobile 脚手架
- **统一插件模式**：移除 HTML 插件模式，仅保留 React/TSX 模式
- 更新 CI release workflow 路径引用

## ✨ 新功能

### 项目系统
- 基于项目的独立工作区，每个项目独立上下文
- 项目级配置文件（Settings），支持项目级别的 Skills、Agents、MCP Servers 等配置
- 项目列表管理，支持创建、切换、删除项目

### 工具箱升级
- MCP Gateway：支持 SSE、stdio、Streamable HTTP 三种传输协议
- 沙箱执行：JavaScript（含 npm 包）和 Python（含 pip 包）隔离执行
- Agent 编排系统：支持委托任务给自治 Agent
- 文件转换：PDF、DOCX、PPTX、XLSX → Markdown

### 界面全面重构
- 工作区面板：文件树浏览、输出文件管理
- 拖拽上传：支持拖拽文件到聊天输入区
- Token 用量进度条
- 对话压缩（Context Window 管理）
- 控制台面板（工具调用日志）
- 任务列表面板（Agent 任务追踪）

### 记忆系统
- 短期记忆：对话中自动提取
- 长期记忆：跨对话持久化
- Supabase 云端同步（可选）

## 🛠 改进

- CI release 构建优化（平台分离、扩展名匹配）
- 插件权限模型完善
- Electron 主进程稳定性提升
