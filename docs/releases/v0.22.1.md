# Stardust v0.22.1

发布日期：2026-06-14

## 🔄 Agent 输出重构

- **时间线穿插显示**：文字和工具调用按实际发生顺序自然穿插，不再分离显示
- **结构化渲染**：工具调用改为组件渲染（Tailwind 样式卡片），支持展开/折叠输出
- **成对匹配**：调用和返回精确配对，输出紧跟在对应调用下方
- **入参摘要**：工具调用显示关键入参，同名工具可区分
- **彩色状态**：执行中（黄色）、完成（绿色 ✓）、失败（红色 ✗）

## 🎨 界面改进

- **控制台 emoji 保留**：消息区 emoji 替换为彩色圆点，控制台保持 emoji
- **Agent 工卡**：消息头部右侧 IdCard 图标，点击查看 Agent 详情（Skills、类型、Endpoint）
- **聊天链接**：点击链接用系统浏览器打开，不再内嵌窗口
- **关于页优化**：版本胶囊、时间线更新日志、工作室官网链接
- **图标更新**：登录页/侧边栏改用 SVG Logo，白/黑双色适配

## 🔧 功能改进

- **从 GitHub 安装 Skill**：支持 `git clone` 仓库或子目录安装
- **Agent/Skill 开关**：聊天中每个 Agent 和 Skill 可独立启用/停用
- **任务进度改善**：AI 回复结束自动完成剩余任务，任务可手动点击切换
- **工具箱编辑**：分类重命名、工具编辑
- **分类名/工具编辑**：支持修改名称和属性

## 🛠 修复

- 版本号统一从 package.json 读取
- 社区插件中文乱码问题
- GitHub API 403 限流改为 jsDelivr CDN
- 链接外部打开（shell.openExternal）
