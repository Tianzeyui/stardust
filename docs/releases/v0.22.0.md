# Stardust v0.22.0

发布日期：2026-06-14

## 🏪 社区插件

- **社区插件浏览安装**：从 stardust-community-plugins 仓库浏览和安装插件
- 通过 jsDelivr CDN 分发，零 GitHub API 依赖，国内可访问
- 5 分钟本地缓存，离线可查看
- 安装进度实时显示
- 已安装插件自动标记

## 📦 插件仓库管理

- **Git 仓库作为插件源**：支持添加任意 Git 仓库（含私有仓库）
- 自动 `git clone --depth 1` 到本地
- 支持 `git pull` 刷新仓库
- 从本地克隆目录直接安装，无需网络下载
- 需安装 Git，私有仓库需配置 SSH Key 或凭据

## 🔧 改进

- **Skill 开关**：聊天中每个 Skill 可独立启用/禁用
- **中英文双语 README**：顶部导航切换，GitHub 默认显示英文
- **社区插件迁移**：`_template`、`diary`、`hello-world`、`inspiration` 移至独立仓库
